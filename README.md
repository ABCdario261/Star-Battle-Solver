# Star Battle Solver (Prolog)

Este projeto consiste na implementação de um solver do puzzle "Star Battle" em Prolog, para a cadeira Lógica para programação

O objetivo do programa é resolver automaticamente tabuleiros do puzzle, respeitando 
todas as regras do jogo através de raciocínio lógico e programação declarativa.

## Descrição do problema

O puzzle "Star Battle" consiste num tabuleiro NxN dividido em várias regiões. 
O objetivo é colocar um número fixo de estrelas de forma que:

- Cada linha contém exatamente 2 estrelas
- Cada coluna contém exatamente 2 estrelas
- Cada região contém exatamente 2 estrelas
- Nenhuma estrela pode estar adjacente a outra (incluindo diagonais)

O programa recebe um tabuleiro inicial e aplica regras lógicas para encontrar uma solução válida.

## Tecnologias e conceitos
- Linguagem: Prolog
- Programação declarativa
- Resolução de restrições 
- Manipulação de listas

## Funcionalidades
- Representação de tabuleiros NxN
- Validação de regras do puzzle
- Colocação automática de estrelas
- Resolução de puzzles completos
